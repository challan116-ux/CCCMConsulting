# Image assets

Drop real imagery here. Referenced by `index.html`.

## Required

- **`signaledi-hero.webp`** — the SignalEDI product screenshot used as the hero
  for the SignalEDI work card. If this file is missing, the card falls back to
  the built-in SVG mockup automatically.

  Recommended: a wide screenshot (≥ 1600px wide), `.png` or `.webp`. The card
  crops to roughly a 2:1 ratio, focused near the top of the image.

## Tips

- Convert to **WebP** for smaller files (`signaledi-hero.webp`) and update the
  `img:` path in the `WORK` array in `index.html`.
- Keep real, identifiable product UI here; use the SVG mockups for the template
  projects (Vantage, Beacon, Atlas).
